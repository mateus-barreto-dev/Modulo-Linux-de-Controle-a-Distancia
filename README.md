# Modulo-Linux-de-Controle-a-Distancia
Implementação de um Módulo Linux que permita o controle do Desktop a distância, utilizando o celular


Funcionalidades atuais:
- Movimento do mouse
- Clique esquerdo/direito
- Scroll
- Pressionar e liberar teclas
- Comunicação TCP
- Envio de Imagens da Tela do Computador para o Celular
- Aplicativo Android em desenvolvimento parcialmente completo
    1.  Desenvolvimento do Controle Remoto do Mouse Concluído
    2.  Desenvolvimento do Controle Remoto do Teclado (Concluído com ressalvas de que nem todas as teclas funcionam, mas as principais Shift, Crtl, Alt esquerdos, Tab, Backspace e Espaço funcionam)
    3.  Desenvolvimento da Recepção da Tela do Computador (Atual/Parcialmente desenvolvido -> Atualmente diminuindo a quantidade de pacotes enviados pela rede, mas já funcional)
