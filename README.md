# Modulo-Linux-de-Controle-a-Distancia
Implementação de um Módulo Linux que permita o controle do Desktop a distância, utilizando o celular


Funcionalidades atuais:
- Movimento do mouse
- Clique esquerdo/direito
- Scroll
- Pressionar e liberar teclas
- Digitação de texto
- Comunicação TCP
- Aplicativo Android em desenvolvimento
